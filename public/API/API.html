API

Denne side omhandler hvordan man kan bruge HTTP metoder til at se, indsætte, redigere og slette i et API. Der er brugt en json() metode for at kunne bruge JSON filer, en urlencoded() metode for at kunne bruge det vi tager fra url'en og der er brugt en static() metode for at kunne tilgå js, html og css filer fra en public mappe



Knappen 'Vis alle biler' viser alle de biler der er i listen med JSON objekter

I tekstfeltet Bil kan der indtastes navnet på en bil, som man herefter kan tilføjes til listen over JSON objekter ved at bruge knappen 'Tilføj bil'. Bilen man tilføjer bliver automatisk tildelt et ID

Hvis man også tilføjer et ID i ID tekstfeltet, og bruger knappen 'Rediger bil', kan man redigere bilen med det specificerede ID

Hvis man kun bruger ID tekstfeltet, og trykker på knappen 'Find med ID', får man vist bilen der ligger på det specificerede ID

Hvis man kun bruger ID tekstfeltet og trykker på knappen 'Slet bil', så sletter man bilen på det specificerede ID

Hold musen henover en knap for at se yderligere informationer
